👾 Death Monster

Your worst run is your best teacher.

Death Monster is a 2D meta-platformer where failure helps you progress. Every time the player dies, the level evolves by turning past mistakes into platforms needed to reach the goal.

🕹️ Concept

Unlike traditional games that punish failure, Death Monster rewards persistence. Each Game Over changes the world, allowing players to eventually cross obstacles using the paths created by their previous runs.

🧬 Level Evolution

0 Deaths – Barren State: Impossible gaps and no help.

1–3 Deaths – Echoes: Transparent ghosts replay past movements.

3+ Deaths – Solid Platforms: Ghosts become jumpable platforms.

5+ Deaths – Shortcut: A hidden golden path appears.

8+ Deaths – Morph: Level geometry shifts to help the player.

10+ Deaths – Bridge of Martyrs: A full ghost pathway leads to the goal.

🛠️ Features

Path memory system that records player movement.

Ghost actors created after each respawn.

Dynamic level changes based on death count.

UI Persistence Tracker showing learned “lessons”.
